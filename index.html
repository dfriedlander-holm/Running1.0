Strava-powered running analytics dashboard for GitHub Pages.

Data Source

Paste a short-lived access token (not an authorization code). If you see a 401, the token is usually expired, missing run/activity scopes, or copied incorrectly. Tokens are never stored.

Create a token in your own Strava OAuth app and paste it here. Tokens are never stored.

CSV columns accepted: date, distance_mi (or meters/km), and optional moving_time_sec.
