Running1.0 Tracker

Strava-powered running analytics dashboard for GitHub Pages.

Data Source

Paste either a Strava access token or authorization code. Authorization code exchange requires the local backend endpoint (/api/strava/exchange) to be running with your Strava client credentials.

CSV columns accepted: date, distance_mi (or meters/km), and optional moving_time_sec.

Targets

Key Metrics

Weekly Breakdown Remaining

Monthly Comparison

Pace Analysis

7-Day Rolling Mileage

Distance Histogram (Current Year)

This is a client-side app. Keep your Strava token private and temporary.
